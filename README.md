## Installation
```
$ git clone --recurse-submodules https://github.com/ahokcool/cub3D.git    # Clone
$ cd cub3D                                                                # Change directory
$ make                                                                    # Compile
$ ./cub3D <map_file.cub>                                                  # Run
```
